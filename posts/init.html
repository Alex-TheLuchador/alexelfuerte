Welcome

I'm Alex Hernandez - an IT technician (at the time of writing...) who is passionate about cloud technology. I have professional experience is in Azure, though I have poked around in AWS in my free time.

I really like Azure's integration with other Microsoft products, and joining devices to Azure AD seems to be very useful. I haven't looked deep enough into AWS to know if there is a similar solution there.

My intention with this blog is to help me gain an understanding of Azure as I study for the AZ-104 exam, as well as become a resource for others as they break into cloud and perhaps other fields in tech. I believe that writing things out knowing that someone will read my work to learn Azure and cloud will force my understanding to be greater, so that I can make be a useful resource for others.

I hope you find my blog posts to be useful and enjoyable.
